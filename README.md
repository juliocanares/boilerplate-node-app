# MVC Node.js application boilerplate

> A boilerplate for **Node.js** running on top of **Express.js**.

**Demo:** it's coming

## Directory Structure

* models
* views
* controllers
* routes


## Featured Included
	
* mvc structure
* jobs management
* sign up
* login
* logout
* email validation
* password recovery
* etc.

## Libraries Included

* express.js
* sequelize.js
* crypto.js
* passport.js

## Instructions

Use with ``nodejs^0.12.0`, clone the repo, `npm install` and `node app.js`.
